# 1306. Jump Game III（跳跃游戏 III）

- **难度**: Medium
- **标签**: DFS, BFS, Array
- **日期**: 2026-05-19
- **耗时**: 2ms | 59.4MB

## 思路

图的可达性问题。从 start 出发，每步可跳 i±arr[i]，问能否到达任一 arr[i]=0 的节点。

### 最优解：正向 BFS + 原地标记

- 从 start 出发 BFS
- 已访问的节点标记为 -1（原地修改，省掉 visited set）
- 碰到 arr[i]=0 直接返回 true

## 代码

```typescript
function canReach(arr: number[], start: number): boolean {
    const n = arr.length;
    const queue: number[] = [start];

    while (queue.length > 0) {
        const i = queue.pop()!;
        if (arr[i] === 0) return true;
        if (arr[i] < 0) continue;

        const step = arr[i];
        arr[i] = -1;

        if (i + step < n) queue.push(i + step);
        if (i - step >= 0) queue.push(i - step);
    }

    return false;
}
```

## 复杂度

- 时间：O(n) — 每个节点最多访问一次
- 空间：O(n) — queue 最坏情况

## 备注

- 用户初始思路是建反向图（从 0 节点出发 BFS 看能否到 start），等价但代码量多一倍
- 原地标记 arr[i]=-1 省掉了 visited Set，内存 beat 100%
