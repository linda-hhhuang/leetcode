# LeetCode #2812 - 找出最安全路径

- **难度**: Medium
- **日期**: 2026-07-02 AC
- **语言**: TypeScript
- **Runtime**: 1089ms | Memory: 93.9 MB

## 题意

n×n 网格有小偷(1)和空格(0)，从(0,0)走到(n-1,n-1)，路径安全因子=路径上所有格子到最近小偷的曼哈顿距离的最小值。求所有路径中最大的安全因子。

## 思路

两步走：
1. **多源 BFS** 建距离矩阵：所有小偷同时入队扩散，每格第一次被访到的层数 = 到最近小偷的距离。O(n²)
2. **二分答案** + BFS 验证连通性：二分安全因子 mid，验证"只走 dist≥mid 的格子能否从起点到终点"。O(n² log n)

关键洞察：四方向网格上 BFS 步数 = 曼哈顿距离。

## 复杂度

- 时间: O(n² log n)
- 空间: O(n²)

## 代码

```typescript
function maximumSafenessFactor(grid: number[][]): number {
    const n = grid.length;
    const dirs = [[1,0],[-1,0],[0,1],[0,-1]];
    const dist: number[][] = Array.from({length: n}, () => new Array(n).fill(-1));
    const queue: [number, number][] = [];
    for (let r = 0; r < n; r++)
        for (let c = 0; c < n; c++)
            if (grid[r][c] === 1) { dist[r][c] = 0; queue.push([r, c]); }
    let qi = 0;
    while (qi < queue.length) {
        const [r, c] = queue[qi++];
        for (const [dr, dc] of dirs) {
            const nr = r + dr, nc = c + dc;
            if (nr >= 0 && nr < n && nc >= 0 && nc < n && dist[nr][nc] === -1) {
                dist[nr][nc] = dist[r][c] + 1;
                queue.push([nr, nc]);
            }
        }
    }
    if (dist[0][0] === 0 || dist[n-1][n-1] === 0) return 0;
    const canReach = (mid: number): boolean => {
        if (dist[0][0] < mid || dist[n-1][n-1] < mid) return false;
        const vis = Array.from({length: n}, () => new Array(n).fill(false));
        const q: [number, number][] = [[0, 0]];
        vis[0][0] = true;
        let i = 0;
        while (i < q.length) {
            const [r, c] = q[i++];
            if (r === n-1 && c === n-1) return true;
            for (const [dr, dc] of dirs) {
                const nr = r + dr, nc = c + dc;
                if (nr >= 0 && nr < n && nc >= 0 && nc < n && !vis[nr][nc] && dist[nr][nc] >= mid) {
                    vis[nr][nc] = true;
                    q.push([nr, nc]);
                }
            }
        }
        return false;
    };
    let lo = 0, hi = n + n - 2;
    while (lo < hi) {
        const mid = (lo + hi + 1) >> 1;
        if (canReach(mid)) lo = mid;
        else hi = mid - 1;
    }
    return lo;
}
```

## 关键洞察

- "每个点到最近一组源点的距离" → **多源 BFS**（不是逐个枚举 O(n⁴)）
- "最大化路径上的最小值" → **二分答案 + 连通性验证**，这是固定模式
- 二分的 check 用 BFS 判「只走 ≥mid 的格子能否到终点」
- 四方向网格上 BFS 步数恰等于曼哈顿距离
