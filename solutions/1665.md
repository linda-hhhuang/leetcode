# LeetCode #1665 - 完成所有任务的最少初始能量

- **难度**: Hard
- **日期**: 2026-05-13 AC
- **语言**: TypeScript
- **Runtime**: 32ms | Memory: 85.3MB
- **Submission ID**: 724781376

## 题意

给定 tasks[i] = [actual_i, minimum_i]，actual 是完成任务消耗的能量，minimum 是开始任务所需的最低能量。求完成所有任务所需的最少初始能量。

## 思路

1. **贪心排序**：按 C = (minimum - actual) 降序排列
   - 交换论证：C 大的放前面，能让后续任务的门槛更容易满足
2. **反向 DP**：从后往前算每个位置的最低初始能量
   - `Mn = Bn`
   - `Mk = max(Bk, Ak + Mk+1)`
   - `M1` 即为答案

## 复杂度

- 时间: O(n log n)（排序主导）
- 空间: O(1)（原地排序 + 单变量）

## 代码

```typescript
function minimumEffort(tasks: number[][]): number {
    tasks.sort((a, b) => (b[1] - b[0]) - (a[1] - a[0]));

    const n = tasks.length;
    let m = tasks[n - 1][1];
    for (let i = n - 2; i >= 0; i--) {
        m = Math.max(tasks[i][1], tasks[i][0] + m);
    }

    return m;
}
```

## 关键洞察

- 排序 tiebreaker 不影响结果（C 相等时任意顺序等价）
- 这是理论最优解（比较排序下界 Ω(n log n)）
