# LeetCode #3121 - 统计特殊字母的数量 II

- **难度**: Medium
- **日期**: 2026-05-28 AC
- **语言**: TypeScript
- **Runtime**: 23ms | Memory: 63.6 MB

## 题意

字母 c 是"特殊"的：小写和大写都出现，且所有小写都在第一个大写之前。

## 思路

单数组 state[26] 做状态机，4 种状态单遍扫描：
- 0: 未出现
- 1: 只见过小写
- 2: 小写后见到大写（合法终态）
- -1: 非法（冻结）

状态转移：
| 当前 | 遇小写 | 遇大写 |
|------|--------|--------|
| 0 | → 1 | → -1 |
| 1 | → 1 | → 2 |
| 2 | → -1 | → 2 |
| -1 | 冻结 | 冻结 |

## 复杂度

- 时间: O(n)
- 空间: O(1)（固定 26 个槽位）

## 代码

```typescript
function numberOfSpecialChars(word: string): number {
    const state = new Array(26).fill(0);
    for (const ch of word) {
        const code = ch.charCodeAt(0);
        if (code >= 97) {
            const k = code - 97;
            if (state[k] === 0) state[k] = 1;
            else if (state[k] === 2) state[k] = -1;
        } else {
            const k = code - 65;
            if (state[k] === 1) state[k] = 2;
            else if (state[k] === 0) state[k] = -1;
        }
    }
    let count = 0;
    for (let i = 0; i < 26; i++) {
        if (state[i] === 2) count++;
    }
    return count;
}
```

## 关键洞察

- 设计状态机必须枚举**每个状态×每种输入**的完整网格
- 关键易错点：state=2 遇到大写应保持 2（多个大写不影响），state=2 遇到小写才是非法
- 比记录 lastLower/firstUpper 位置的解法更优雅，只需一遍扫描无需预处理
