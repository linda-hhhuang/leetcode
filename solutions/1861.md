# 1861. Rotating the Box（旋转盒子）

- **难度**: Medium
- **标签**: Array, Two Pointers, Matrix
- **日期**: 2026-05-14
- **耗时**: 16ms | 80.1MB

## 思路

两步走：
1. **重力模拟**：旋转后右边变成下方，所以先在原矩阵每行让石头"向右落"
2. **旋转**：`res[i][j] = box[m-1-j][i]`

### 重力模拟的 k 计数器技巧

遍历每行，维护 `k` = 当前连续石头组的大小：
- 遇到 `*`：k=0（障碍物截断）
- 遇到 `.`：如果 k>0，swap(arr[x], arr[x-k])，石头组整体右移一格
- 遇到 `#`：k++

核心洞察：swap 后 k 不变，因为石头组大小没变，只是位置平移了。

## 代码

```typescript
function rotateTheBox(boxGrid: string[][]): string[][] {
    const m = boxGrid.length;
    const n = boxGrid[0].length;

    for (let r = 0; r < m; r++) {
        let k = 0;
        for (let x = 0; x < n; x++) {
            if (boxGrid[r][x] === "*") {
                k = 0;
            } else if (boxGrid[r][x] === ".") {
                if (k > 0) {
                    boxGrid[r][x] = "#";
                    boxGrid[r][x - k] = ".";
                }
            } else {
                k++;
            }
        }
    }

    const res: string[][] = Array.from({ length: n }, () => new Array(m));
    for (let i = 0; i < n; i++) {
        for (let j = 0; j < m; j++) {
            res[i][j] = boxGrid[m - 1 - j][i];
        }
    }

    return res;
}
```

## 复杂度

- 时间：O(m×n)
- 空间：O(m×n)（输出矩阵，原地模拟不需要额外空间）

## 最优性

O(m×n) 是最优——需要读写每个格子至少一次。
