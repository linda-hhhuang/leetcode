# LeetCode #3120 - 统计特殊字母的数量 I

- **难度**: Easy
- **日期**: 2026-05-26 AC
- **语言**: TypeScript
- **Runtime**: 3ms | Memory: 58.8 MB

## 题意

字符串中同时出现大小写的字母称为"特殊字母"，统计其数量。

## 思路

1. 两个长度 26 的数组 `low`、`up`，遍历字符串标记出现情况
2. 最后遍历 26 位，`low[i] && up[i]` 计数

## 复杂度

- 时间: O(n)
- 空间: O(1)（固定 52 个槽位）

## 代码

```typescript
function numberOfSpecialChars(word: string): number {
    const low = new Array(26).fill(0);
    const up = new Array(26).fill(0);
    for (const x of word) {
        const code = x.charCodeAt(0);
        const kLow = code - 97;
        const kUp = code - 65;
        if (kLow >= 0 && kLow <= 25) low[kLow] = 1;
        if (kUp >= 0 && kUp <= 25) up[kUp] = 1;
    }
    let sum = 0;
    for (let i = 0; i < 26; i++) {
        sum += (up[i] && low[i]) ? 1 : 0;
    }
    return sum;
}
```

## 关键洞察

- 送分题，无陷阱
- 也可以用一个 52-bit 位掩码或 Set 实现，但双数组最直观
