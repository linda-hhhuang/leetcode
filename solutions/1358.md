# LeetCode #1358 - 包含所有三种字符的子字符串数目

- **难度**: Medium
- **日期**: 2026-06-30 AC
- **语言**: TypeScript
- **Runtime**: 11ms | Memory: 57.2 MB

## 题意

只含 a/b/c 的字符串，求同时包含至少一个 a、b、c 的子串数目。

## 思路

滑动窗口，按 right 累加：
1. cnt[3] 记录窗口内 abc 数量
2. 固定 right，当窗口含全部 abc 时尽量右移 left（移到再移一步就不合法）
3. 以 right 结尾的合法子串左端点可取 0..left-1，共 left 个
4. 累加 left

## 复杂度

- 时间: O(n)（left/right 各单向移动）
- 空间: O(1)

## 代码

```typescript
function numberOfSubstrings(s: string): number {
    const cnt = [0, 0, 0];
    let left = 0, result = 0;
    for (let right = 0; right < s.length; right++) {
        cnt[s.charCodeAt(right) - 97]++;
        while (cnt[0] > 0 && cnt[1] > 0 && cnt[2] > 0) {
            cnt[s.charCodeAt(left) - 97]--;
            left++;
        }
        result += left;
    }
    return result;
}
```

## 关键洞察

- 双指针单向移动是 O(n) 的关键，不是 O(n²)
- 两种对偶累加方式：按 left 累加 (n - 最小合法 right) 或按 right 累加 (合法左端点数 = left)，结果一致
- "至少包含" 类计数题的通用套路：固定一端，找另一端的临界点，区间内的都合法
