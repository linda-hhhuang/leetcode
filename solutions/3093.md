# LeetCode #3093 - 最长公共后缀查询

- **难度**: Hard
- **日期**: 2026-06-22 AC
- **语言**: TypeScript
- **Runtime**: 1477ms | Memory: 206.3 MB

## 题意

给 wordsContainer 和 wordsQuery，对每个 query 找 container 中最长公共后缀对应的字符串（优先短、再优先下标小）。

## 思路

1. 将 wordsContainer 的每个字符串**从尾到头**逐字符插入 Trie（等效于反转后建前缀 Trie）
2. 每个 Trie 节点维护 `bestIdx`：经过该节点的所有字符串中，长度最短且下标最小的
3. 查询时将 query 从尾到头走 Trie，走到走不动为止，最后到达的节点的 bestIdx 即答案
4. root 节点存全局 best（所有字符串中最短的），作为无任何后缀匹配时的 fallback

## 复杂度

- 时间: O(Σ|wordsContainer[i]| + Σ|wordsQuery[i]|)
- 空间: O(Σ|wordsContainer[i]|) Trie 节点

## 代码

```typescript
function stringIndices(wordsContainer: string[], wordsQuery: string[]): number[] {
    const root: [Map<number, any>, number] = [new Map(), 0];

    let globalBest = 0;
    for (let i = 1; i < wordsContainer.length; i++) {
        if (wordsContainer[i].length < wordsContainer[globalBest].length) {
            globalBest = i;
        }
    }
    root[1] = globalBest;

    for (let i = 0; i < wordsContainer.length; i++) {
        const w = wordsContainer[i];
        let node = root;
        for (let j = w.length - 1; j >= 0; j--) {
            const c = w.charCodeAt(j) - 97;
            if (!node[0].has(c)) {
                node[0].set(c, [new Map(), i]);
            }
            node = node[0].get(c)!;
            const curBest = node[1];
            if (wordsContainer[i].length < wordsContainer[curBest].length ||
                (wordsContainer[i].length === wordsContainer[curBest].length && i < curBest)) {
                node[1] = i;
            }
        }
    }

    const ans: number[] = [];
    for (const q of wordsQuery) {
        let node = root;
        let best = root[1];
        for (let j = q.length - 1; j >= 0; j--) {
            const c = q.charCodeAt(j) - 97;
            if (!node[0].has(c)) break;
            node = node[0].get(c)!;
            best = node[1];
        }
        ans.push(best);
    }

    return ans;
}
```

## 关键洞察

- "最长公共后缀" = 反转后的"最长公共前缀"，直接复用 Trie 思路
- 不需要存排序列表，每个节点只存一个 best 即可（因为查询只关心最优解）
- 无需显式反转字符串，直接从 `w.length-1` 到 `0` 遍历等效于反转
- Map 作为 children 在 JS/TS 中虽然比定长数组慢，但题目只有 26 个字母，影响不大
